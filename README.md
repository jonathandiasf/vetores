Código em C para registrar e somar vendas usando vetores em um período de 5 dias.
## Como Usar ##

1. Instale um compilador C
  - No Windows: instale o MinGW ou use o compilador do Visual Studio Code.
  - No Linux/Mac: geralmente o gcc já vem instalado ou pode ser instalado via terminal( sudo apt installbuild-essential no Ubuntu).

2. Crie o arquivo do código
   - Abra um editor de texto(VS Code, Notepad++, Bloco de Notas...), copie o código e salve como vetores.c

3. Compile o código
   - Abra o terminal ou prompt de comando.
   - Navegue até a pasta onde está o arquivo
   - Digite:
    gcc vendas.c -o vendas
   - Isso gera um executável chamado vetores.

4. Execute o programa
   - No Linux/Mac
     ./vendas
   - No Windows
     vendas.exe
5. Interaja com o programa
   - Digite valores para cada dia solicitado, faça testes com diferentes entradas
   - O programa exibirá os valores registrados e o total do período.
